# LatihanSoalUKL3No1
Fungsi Utama: Program ini berfungsi untuk menghitung nilai rata-rata (average) dari sejumlah nilai yang dimasukkan oleh user.
Cara Kerja: Program pertama-tama meminta jumlah total siswa yang nilainya akan dihitung,Menggunakan perulangan (for), program akan meminta nilai satu per satu sebanyak jumlah siswa tersebut, sambil menjumlahkan semua nilai yang dimasukkan ke dalam variabel,Setelah semua nilai terkumpul,dan program akan membagi totalNilai dengan jumlahSiswa untuk mendapatkan rata-rata,Hasil rata-rata (misalnya, 85.0) kemudian ditampilkan kepada user.
